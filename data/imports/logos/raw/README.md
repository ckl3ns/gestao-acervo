# Exportações reais do Logos

Esta pasta existe para armazenar exportações reais do Logos usadas em validação manual.

## Arquivos esperados

Exemplo de convenção:
- `Biblioteca-Logos-10(YYYYMMDD).csv`
- `Biblioteca-Logos-7(YYYYMMDD).csv`

## Regras

- não comitar exportações completas se elas forem grandes ou privadas;
- manter nesta pasta apenas arquivos locais de trabalho;
- usar `data/samples/` para fixtures pequenas e comitáveis;
- atualizar `docs/formats/logos_csv.md` quando uma nova variante estrutural aparecer.
